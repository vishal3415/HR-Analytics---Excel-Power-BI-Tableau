# HR-Analytics---Excel-Power-BI-Tableau
End-to-end HR Analytics project developed in Power BI and Excel, featuring employee attrition analysis, department-wise insights, job satisfaction metrics, gender diversity analysis, and KPI dashboards to help organizations make strategic HR decisions.
# HR Analytics Dashboard

## Project Overview
This project analyzes employee data to identify workforce trends, employee attrition patterns, and key HR metrics using Power BI.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Key Insights
- Employee Attrition Analysis
- Department-wise Performance
- Gender Diversity Metrics
- Job Satisfaction Analysis
- Age Group Distribution
- Employee Retention Trends

## Dashboard Features
- Interactive Filters and Slicers
- KPI Cards
- Trend Analysis
- Dynamic Visualizations
- Drill-through Reports

## Business Objective
To help HR teams make data-driven decisions regarding employee retention, workforce planning, and organizational performance.
